# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this extension, please report it privately:

- **Email:** <d310151c2@gmail.com>
- **Preferred format:** Steps to reproduce, expected behavior, actual behavior, and related logs if available.

Please avoid public disclosure until we have confirmed and released a fix.

---

## Supported Versions

We provide security updates for the following versions:

| Version | Supported |
| ------- | --------- |
| latest  | ✔️ |
| older releases | ❌ |

Security patches are released only for the latest version.
